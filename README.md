# Gaurav Kumar Rai — Portfolio

A premium personal portfolio built with **Next.js**, **React**, and **Tailwind CSS**.

## Features

- Modern, responsive design (mobile-first)
- Dark mode with toggle
- Smooth animations and interactive effects
- Sections: Hero, About, Skills, Education, Experience, Projects, Certifications, Contact
- Contact form with mailto integration
- SEO optimised metadata
- WCAG accessible

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

The easiest deployment option is [Vercel](https://vercel.com) — import the GitHub repository and it deploys automatically.

## Tech Stack

- [Next.js](https://nextjs.org/) 16
- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/) 4
- TypeScript
